
# 🏎️ F1 Manager

Un joc de mànager de Fórmula 1 fet amb HTML, CSS i JavaScript vanilla. Gestiona la teva escuderia, contracta pilots, millora el teu equip i competeix en circuits reals!

---

## 🚦 Característiques actuals

- **Sistema d’autenticació**: Registre i login amb localStorage.
- **Gestió d’escuderia**: Crea i personalitza el teu equip.
- **HQ (Headquarters)**: Millora motor, aerodinàmica i xassís.
- **Tenda**: Contracta pilots i mànagers amb habilitats i preus diferents.
- **Equip**: Consulta els teus pilots i estadístiques.
- **Sistema de curses**:  
  - Circuits disponibles: Monaco 🇲🇨, Spa 🇧🇪, Monza 🇮🇹, Portimão 🇵🇹, Interlagos 🇧🇷  
  - Selecció de circuit i estratègia de neumàtics (tous, mitjans, durs).
  - Animació de pilots en temps real.
  - Classificació i premis per posició.
- **Disseny modern**: Interfície responsiva i atractiva, inspirada en la F1.

---

## 🎮 Com jugar

1. **Crea el teu compte** i escuderia.
2. **Contracta pilots i mànager** a la tenda.
3. **Millora el teu HQ** per tenir més rendiment.
4. **Selecciona un circuit** i tria estratègia de neumàtics.
5. **Competeix i guanya** punts i diners per millorar el teu equip!

---

## 🏆 Sistema de puntuació i premis

- 1r lloc: 25 punts + €1M
- 2n lloc: 18 punts + €950K
- 3r lloc: 15 punts + €900K
- Altres posicions: punts i premis decreixents
