🎮 MSManager

MSManager es un proyecto de videojuego enfocado en ofrecer una experiencia dinámica donde el jugador gestiona, organiza y toma decisiones estratégicas dentro de un sistema interactivo.

El objetivo principal del juego es poner al jugador en el centro de la gestión, permitiéndole controlar diferentes aspectos del entorno mientras optimiza recursos, mejora su rendimiento y progresa a lo largo de la partida.

🚀 Características
🎯 Sistema de gestión interactivo
⚙️ Mecánicas enfocadas en toma de decisiones
📊 Progresión basada en optimización y estrategia
🧩 Diseño modular que facilita futuras ampliaciones
🎮 Experiencia adaptable según el estilo del jugador
🕹️ Gameplay

En MSManager, el jugador asume el rol de gestor, donde deberá:

Administrar recursos disponibles
Tomar decisiones estratégicas en tiempo real o por turnos
Mejorar sistemas y desbloquear nuevas opciones
Adaptarse a diferentes situaciones y desafíos

Cada decisión influye directamente en el desarrollo del juego, creando una experiencia única en cada partida.

🏗️ Estructura del proyecto

El proyecto está organizado de forma modular para facilitar su mantenimiento y escalabilidad:

Core → Lógica principal del juego
Systems → Mecánicas y sistemas de gestión
UI → Interfaz de usuario
Assets → Recursos del juego (no incluidos aquí)
🧠 Objetivo del proyecto

Este proyecto ha sido desarrollado como:

Experimento de diseño de mecánicas de gestión
Práctica de desarrollo de videojuegos
Base para futuras expansiones o mejoras
