# 🏎️ MS Manager

Un juego de gestión de automovilismo deportivo hecho con HTML, CSS, JavaScript vanilla y backend con Node.js + MongoDB. Gestiona tu escudería, contrata pilotos, mejora tu equipo y compite en circuitos reales de diferentes categorías del motorsport!

---

## 🚦 Características actuales

- **Sistema de autenticación**: Registro e inicio de sesión con backend seguro (JWT + MongoDB Atlas).
- **Gestión de escudería**: Crea y personaliza tu equipo de motorsport.
- **HQ (Headquarters)**: Mejora motor, aerodinámica y chasis.
- **Tienda**: Contrata pilotos y managers con habilidades y precios diferentes.
- **Equipo**: Consulta tus pilotos y estadísticas.
- **Sistema de carreras**:  
  - Circuitos disponibles: Monaco 🇲🇨, Spa 🇧🇪, Monza 🇮🇹, Portimão 🇵🇹, Interlagos 🇧🇷  
  - Selección de circuito y estrategia de neumáticos (blandos, medios, duros).
  - Animación de pilotos en tiempo real.
  - Clasificación y premios por posición.
- **Backend con MongoDB Atlas**: Tus datos se guardan en la nube y están disponibles desde cualquier dispositivo.
- **Diseño moderno**: Interfaz responsive y atractiva, inspirada en el mundo del motorsport.

---

## 🎮 Cómo jugar

1. **Crea tu cuenta** y escudería.
2. **Contrata pilotos y manager** en la tienda.
3. **Mejora tu HQ** para tener más rendimiento.
4. **Selecciona un circuito** y elige la estrategia de neumáticos.
5. **Compite y gana** puntos y dinero para mejorar tu equipo!

## 🗄️ Tecnologías utilizadas

### Frontend
- HTML5
- CSS3
- JavaScript (Vanilla)
- Three.js (intro 3D)

### Backend
- Node.js
- Express.js
- MongoDB Atlas (base de datos en la nube)
- JWT (autenticación)
- bcryptjs (encriptación de contraseñas)
