# 🏎️ F1 Manager - Joc de Gestió

Un joc de mànager de Fórmula 1 desenvolupat amb HTML, CSS i JavaScript vanilla.

## 📋 Característiques

### ✅ Implementades
- **Sistema de comptes**: Registre i login amb localStorage
- **Gestió d'escuderia**: Crea i gestiona la teva pròpia escuderia
- **HQ (Headquarters)**: Millora motor, aerodinàmica i xassís
- **Tenda**: Contracta pilots i mànagers
- **Equip**: Visualitza els teus pilots i estadístiques
- **Sistema de curses**: 
  - 3 circuits diferents (Monaco, Spa, Monza)
  - Estratègies de neumàtics (tous, mitjans, durs)
  - Animació de pilots en temps real
  - Sistema de posicions
  - Punts i premis en diners

### 🎮 Com jugar

1. **Crear compte**: Registra't amb un nom d'usuari i nom d'escuderia
2. **Contractar personal**: Ves a la tenda i contracta 2 pilots i un mànager
3. **Millorar l'HQ**: Inverteix en millores de motor, aero i xassís
4. **Córrer**: Selecciona un circuit, tria l'estratègia de neumàtics i corre!
5. **Guanyar**: Acumula punts, victòries i diners per millorar el teu equip

## 🚀 Començar

Simplement obre `index.html` en un navegador web modern.

No cal instal·lar res, tot funciona amb HTML, CSS i JavaScript vanilla.

## 💡 Pressupost inicial

- **Diners inicials**: €10M

- **Cost millores**: €500K per millora

- **1r lloc**: 25 punts + €1M
- **2n lloc**: 18 punts + €950K
- **3r lloc**: 15 punts + €900K
- **Altres posicions**: Punts decreixents
